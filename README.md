## Technologies Utilized

### Docker and Postgres
Currently, using Postgres with Docker. Postgres was the database service used in the example

We've had issues connecting the database to MySQL Workbench, but it runs and functions fine otherwise.
DockerHub has been used for testing and pulling images as well as tags for deployment. 

### Render
We currently plan to use Render for uploading a functional image of our web app.
